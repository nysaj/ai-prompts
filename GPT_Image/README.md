# GPT Image

GPT Imageで使用するプロンプト・テンプレートを管理するディレクトリです。

## 目的

- 技術書風で初心者にも分かりやすい図解を作成する
- プロンプトをテンプレート化し、毎回同じ品質で画像を生成できるようにする
- テンプレートを継続的に改善・管理する

## ディレクトリ構成

```text
GPT_Image/
├── CheatSheet.md
└── README.md
```

| ファイル | 内容 |
|----------|------|
| CheatSheet.md | GPT Image用チートシートテンプレート |

## 品質方針

GPT Imageで図解を作成する際は、生成後に以下を確認します。

- 誤字脱字
- 文字化け
- コード・記号の表示崩れ
- コードの色付け異常（二重表示を含む）
- 枠線・図形の色ムラや欠け
- レイアウト崩れ・文字の重なり
- 十分な解像度で文字が読めること
